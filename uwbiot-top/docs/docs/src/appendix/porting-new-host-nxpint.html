9.10. Porting to new Host¶

9.10.1. Layers / Namespace and Convention¶

(Kindly pre read Section 1.2: Acronyms.)

The UWB MW is portable across various platforms and OS.

The following convention is used for the name space.

BUS:

BUS Implementation is the implementation at the Host.

It maps to implementation of SDK Vendor.

For each Host platform a new/different implementation is needed.

UWBS:

This implementation is UWBS Specific.

In order to maintain portability, different uwb_uwbs_tml_io.h is present for each UWBS.

OSAL:

Operating System Specific implementation.

OS Specific implementation like thread, timer, mutexes, semaphores etc. Different header files are present for osal api definitions.
