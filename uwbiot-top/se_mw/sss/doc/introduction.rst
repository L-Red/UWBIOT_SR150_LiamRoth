==================================================================
 SSS: Introduction
==================================================================

``SSS`` is an acronym for **S**\ ecure **S**\ ub **S**\ ystem

The SSS APIs are functional APIs to abstract the access to
various types of Cryptographic Sub Systems.  Such secure
subsystems could be (but not limited to):

- Secure Elements
- Secure Enclaves
- Cryptographic HWs
